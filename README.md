# Emotion Detection Statistical Study

## 📌 Overview
This project focuses on detecting human emotions from facial images using computer vision and statistical analysis. It combines OpenCV with data visualization techniques to provide insights into emotional patterns.

## 🎯 Objectives
- Detect facial emotions using image processing
- Perform statistical analysis on detected emotions
- Visualize emotion distribution using graphs
- Provide an interactive UI in Jupyter Notebook

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Seaborn
- ipywidgets (for UI)
